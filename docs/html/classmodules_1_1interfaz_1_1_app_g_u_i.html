Detailed Description

Clase principal de la interfaz gráfica.

Crea la ventana, los componentes gráficos y gestiona todas las acciones del usuario relacionadas con:

  • Selección de audio
  • Conversión a texto
  • Encriptación y desencriptación
  • Visualización de RAW
  • Guardado de registros
